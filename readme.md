# DANCE-O-MATIC

It's an automated Choreography generator!

Requirements:
- python
- bash
- pyen, the echonest python API
- numpy
- exported ECHO_NEST_API_KEY (for the echo nest api)
- nodejs

to install: `npm install`
to run: `npm run-script debug`
